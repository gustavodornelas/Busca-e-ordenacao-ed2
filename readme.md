# TRABALHO INDIVIDUAL!

Exibir um menu com as seguintes opções:

## a. Gerar vetor
Ler um número e gerar um vetor de inteiros aleatório do tamanho do número lido.

## b. Ordenar vetor
Deve executar apenas se a função da letra a já tiver sido chamada. Ler uma letra (Q ou M) e ordenar de acordo com a letra lida (Quick ou Merge Sort). No final, exibir o vetor ordenado.

## c. Busca sequencial

Deve executar apenas se a função da letra a já tiver sido chamada. Deve ler um número e imprimir se o mesmo foi ou não encontrado no vetor e se encontrado, qual posição.

## d. Busca binária

Deve executar apenas se a função da letra a e da letra b já foi chamada. Deve ler um número e imprimir se o mesmo foi ou não encontrado no vetor e se encontrado, qual posição.
